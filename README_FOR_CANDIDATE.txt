ПРОЕКТ: Telegram-бот для салона красоты Anna Paris
ПОРТ: 8013
ВЕРСИЯ: Сентябрь 2025

СТРУКТУРА:
- main.py - основной файл приложения
- app/ - модули системы
- prompts.yml - промпты для Claude AI
- requirements.txt - зависимости Python
- .env-example - пример переменных окружения

ПЕРЕД ЗАПУСКОМ:
1. Создайте виртуальное окружение: python -m venv .venv
2. Активируйте его: source .venv/bin/activate
3. Установите зависимости: pip install -r requirements.txt
4. Скопируйте .env-example в .env и настройте
5. Добавьте credentials.json для Google API
